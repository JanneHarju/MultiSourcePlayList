Help

  1. If you face any issues try to refresh your browser.
    1. If this doesn't fix issue you counter try to force refresh your browser by pressing shift + refresh button
  2. If spotify won't start to play try to press Change spotify user and confirm dialog by pressing OK.
    1. This might occure if you open app yesterday and spotify token is expired (token expired every 1h, but token is refreshed automatically when app is active)
  3. If spotify still won't start to play try to activate your spotify application by start playing any track.
    1. This application assume that you have spotify application active when you start playing track.
    2. You must have Spotify Premium to control Spotify remotely.

Musiple 0.57

Author Janne Harju

Used components:

  1. .Net Core in backend
  2. Angular (4.1.2) in fronend
  3. Spotify API
  4. Google's Youtube API
  5. ng2-youtube-player for playing YouTube videos
  6. ng2-dnd in drag and drop track ordering
  7. ngx-uploader in file upload. (drag and drop file or select file)
  8. ng2-simple-timer with every timer application needed
  9. MusixMatch API for lyrics
  10. Azure for deploy
  11. Entity Framework with SQL Server

Features:

    1. Spotify
    2. Search
    3. Play track on active spotify device which support Spotify Connect
      1. Application doesn't support yet device selection
    4. Browse spotify artists and albums
    5. Load current spotify users Spotify playlists
      1. User can add whole playlist to local playlist at once
    6. Seek to position
    7. Change spotify user
    8. User must have Spotify premium to use spotify with this application
  2. Youtube
    1. Search
    2. Play (video is hidden only video sound is heard)
  3. Custom playlists
    1. Create playlist which contains tracks from spotify and youtube
    2. Shuffle playlist
    3. Rename playlist
    4. Order playlist by drag and drop
    5. Remove track from playlist
  4. MP3 files
    1. Every user have 1Gb space for own MP3 files. More can be get if needed and asked(for now).
    2. MP3 files are stored at Azure cloud.
    3. MP3 files can be used with any devices (not like in spotify which support only local files)
    4. Only uploader can use own MP3 files. Other users can't see or use other users' files.
  5. MusixMatch
    1. Find lyrics. (find also lyrics for youtube videos if youtube video name is "artisname - trackname" )
  6. Authentication
    1. Login
    2. Register
    3. Remember me
  7. General
    1. Shuffle is real shuffle. (Not like in spotify) So same song can be played again.
